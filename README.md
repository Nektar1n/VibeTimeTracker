# VibeTimeTracker

Vue 3 + Vite приложение для планирования задач и трекинга времени в стиле Apple.

## Возможности

- Календарь на месяц с выбором дня.
- План задач на выбранную дату.
- Добавление задач с проектом и оценкой времени.
- Таймер для конкретной задачи (start/stop, накопление времени).
- Drag & Drop задач между датами в календаре.
- Адаптивный интерфейс для мобильных и широких экранов.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
